CASCADE — Active Learning Evolution Loop

From metagenomic Cas13 hits to predicted allosteric suicide switches. Phase 1 generates both OFF-state (Enzyme + gRNA, dormant) and ON-state (Enzyme + gRNA + target RNA, activated) structures. PXDesign designs linkers only using the OFF-state structure as baseline — so linkers hyper-stabilize the dormant conformation. HEPN domains use expanded boundaries (−60/+100 around catalytic R) with ≥150 residue separation filtering. X residues resolved via baseline lookup or Glycine fallback. RL bias matrix applied post-stitching to steer linker amino acids toward higher fitness. C-terminal tail is always preserved from the wild-type sequence. The global best protein is always used as the next baseline — never regressing. Fitness = HEPN shift + ipTM + AF2-IG − specificity penalties (1/2/3-mismatch off-targets).
